Angular1 example: Tasks Management

Demo

Code in GitHub

This application is developed based on the project of Angular-seed. The idea of it comes from another angular example which I found online. This application can be viewed through the "demo" link. You can also download the world project and run it on your computer. If you try to run it on your computer, install git first. Then make a new dictionary and enter the dictionary. Run "git clone https://github.com/Interest1024/TasksManagement" to download the project. Run "cd TasksManagement" to enter the dictionary "TasksManagement". Run "npm start" to start bower server on your computer. Key in "http://localhost:8000/index.html" in internet explorer to visit the web page.

An example of SpringMVC and RESTful

Code in GitHub

I built the application "SpringMVCREST" following an essay online. This application can provide a RESTful service after it has been delivered by Tomcat. The Spring MVC 4 framework was used to build this application.
File SpringMVCREST/target/SpringMVCREST.war can be copy to webapp2 directory under Tomcat. Then start the Tomcat and key in the URL ( https://localhost:8080/SpringMVCREST/hello) in Chrome. The service will return "hello". Key in the URL ( https://localhost:8080/SpringMVCREST/getAUser) in Chrome. The service will return a user's information in JSON format. You could also use the Postman extension of Chrome to test the application.
All files in my workspace were uploaded into GitHub. The application is under the directory "SpringMVCREST". You could open the whole repository using Eclipse. In the project SpringMVCREST, use maven/updateproject to load all needed jars. Use RunAs/MavenClean and RunAs/MavenInstall to make new war package.
