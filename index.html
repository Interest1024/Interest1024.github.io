Angular1 example: Tasks Management

Demo

Code in GitHub

This application is developed based on the project of Angular-seed. The idea of it comes from another angular example which I found online. This application can be viewed through the "demo" link. You can also download the world project and run it on your computer. If you try to run it on your computer, install git first. Then make a new dictionary and enter the dictionary. Run "git clone https://github.com/Interest1024/TasksManagement" to download the project. Run "cd TasksManagement" to enter the dictionary "TasksManagement". Run "npm start" to start bower server on your computer. Key in "http://localhost:8000/index.html" in internet explorer to visit the web page.

An example of CRUD operations using RESTful web service.

Code in GitHub

This application includes a back-end, which provides RESTful web service and connects to a MySQL database, and a front-end, which provide CRUD operations of Students and Subjects.
This application is developed based on the example of "Spring Boot + AngularJS + Spring Data + JPA CRUD App Example". The example include only CRUD operation of users. I extended the example to support two webpages (CRUD operation of Students and and CRUD operation of Subjects). This project used SpringMVC, Spring Boot, Spring Data, hibernate, AngularJS, ui-router.
This project can be downloaded from GitHub. The SQL scripts for MySQL database are in SQL.txt. The project was run in Eclipse. After downloading the project, you could use Maven/UpdateProject to load needed .jars. Add "-Dspring.profiles.active=prod" to the VM argument in run configuration. Run SpringBootCRUDApp.java as a Java application. Visit the website using "http://localhost:8080/SpringBootCRUDApp/".
