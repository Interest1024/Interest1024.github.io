Angular1 example:

Tasks Management

Demo

Code in GitHub

This application is developed based on the project of Angular-seed. The idea of it come from another angular example which I found online. This application can be viewed through the "demo" link. You can also download the world project into your computer and run it on your computer. If you try to run it on your computer, install git on your computer first. Then make a new dictionary and enter the dictionary. run "git clone https://github.com/Interest1024/TasksManagement" to download the project. Run "cd TasksManagement" to enter the dictionary "TasksManagement". Run "npm start" to start bower server on your computer. Key in "http://localhost:8000/index.html" in internet explorer to visit the web page.
